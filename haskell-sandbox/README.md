# Haskell Sandbox

```shell
stack build --exec "haskell-sandbox-exe example.com pk sk"
```

If you are using Visual Studio Code and the [Haskell GHCi Debug Adapter Phoityne](https://marketplace.visualstudio.com/items?itemName=phoityne.phoityne-vscode) plugin, you may need to restart Visual Studio Code to make the debugger work, see [this comment](https://github.com/phoityne/hdx4vsc/issues/40#issuecomment-3477870880).
